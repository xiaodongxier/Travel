 ## 安装stylus 

 > 类似 less sass 能够声明变量

```shell
sudo cnpm i stylus --save
```

 ## 安装stylus-loader

```shell
sudo cnpm i stylus-loader --save
```

## home 下新建 components

> 存放

##  home 文件引入 HomeHeader.vue 文件

两种写法

```JavaScript
<HomeHeader></HomeHeader>
<home-header></home-header>
```

## style 增加 lang="stylus"

> 使用 stylus 语法

## style 增加 scoped 关键字

> 使之样式文件仅对当前文件生效，对其他文件不生效，避免样式冲突


## 简化 rem 的计算

> 1rem = html font-size = 50px
